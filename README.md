# Blog Project
## Node.js Application
This project allows the user to post.

# Summary
It allows posting for the user who creates an account and logs in.

On the main page, it shows all the posts sent and by whom and on what date these posts were sent.

# Tech Stack
- Node.js
- EJS
- Git
- MongoDB
- Bootstrap
